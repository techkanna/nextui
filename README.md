# @techkanna/nextui

A modern UI component library optimized for Next.js apps.  
Built with TypeScript, Rollup, and Storybook.

## Installation

```bash
npm install @techkanna/nextui
```
or
```bash
yarn add @techkanna/nextui
```

## Usage
```tsx
import { Button } from "@techkanna/nextui";

export default function Home() {
  return <Button>Click me</Button>;
}
```

## Development

```bash
npm run dev      # watch mode
npm run build    # production build
npm run storybook
```
